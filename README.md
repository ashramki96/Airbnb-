## Run the app locally

1. Clone the repository

   `git clone git@github.com:ashramki96/Airbnb-.git`

2. Install dependencies 

   `npm install`

3. Start the backend and front server

   `cd backend` :arrow_right: `npm start` <br>
   `cd frontend` :arrow_right: `npm start`
